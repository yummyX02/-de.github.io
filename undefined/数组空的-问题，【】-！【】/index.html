console.log( [] == ![] ); 为true问题解释

首先 因为 “==”宽松相等，两边操作数类型不一致时，会发生强制类型转换

规则如下：
（1）==两边操作数如果有Boolean类型会转化成数字

（2）有一边为数字，一边为字符串时，则转化为数字进行比较

（3）undefined、null互相转化

（4）有对象类型，先把对象类型转换成基本类型（ToPrimitive），在按以上进行

则比较[] == true，将通过以下 4 个步骤：

  1. [] == true
  2. [] == 1
  3. "" == 1
  4. 0 == 1

而基于运算符优先级，!in![] == true首先执行，因此表达式显然转换为false == trueis false

tips：

值![]是false，因为[]它是一个对象（数组是对象）并且所有对象，不包括null，都是真实的。因此，任何数组，即使它是空的，也始终是真值，而真值的反义词始终是false。检查值是否为真值的最简单方法是使用!!
