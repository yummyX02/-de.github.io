几种控制结构可视的区别

一、html中

display : none 与 visibility:hidden 的区别是什么?

display : 隐藏对应的元素但不挤占该元素原来的空间。

visibility: 隐藏对应的元素并且挤占该元素原来的空间。

即是，使⽤CSS display:none属性后，HTML元素（对象）的宽度、⾼度等各种属性值都将“丢失”;⽽使⽤visibility:hidden属性后，HTML元素（对象）仅仅是在视觉上看不⻅

（完全透明），⽽它所占据的空间位置仍然存在。

二、vue中

v-if 和 v-show 区别？

v-if控制结构不在html结构中存在

而v-show只是视觉上的是否可视 结构中仍是存在的

补充：

v-if实质是动态的创建和销毁DOM元素

• v-show 实质是控制元素的css属性display显⽰和隐藏

使⽤场景: 需要频繁切换的时候使⽤v-show，不需要频繁切换的时候使⽤v-for。这样可以

节省性能开销，使⽤v-if频繁切换会增加性能消耗
